# HackNow OS — Branding

HackNow OS vizual elementlari: GRUB, Plymouth, wallpaper, GTK theme, icon theme, LightDM.

## Struktura

```
grub/              — GRUB bootloader theme
plymouth/          — Plymouth boot splash
wallpapers/        — Desktop wallpaperlar
lightdm/           — Login screen konfiguratsiya
gtk-theme/         — GTK3/4 rang sxemasi
icons/             — HackNow icon theme
neofetch/          — Neofetch ASCII art
```
