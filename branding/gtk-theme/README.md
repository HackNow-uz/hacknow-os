# HackNow Dark GTK Theme

HackNow OS uchun maxsus qora tema. Asosan Adwaita-dark'ga asoslangan,
HackNow rang palitrasi bilan.

## O'rnatish

```bash
cp -r hacknow-dark /usr/share/themes/HackNow-Dark
```

## Ranglar

- Background: #0a0a0a
- Surface: #0f0f10
- Elevated: #111827
- Border: rgba(255,255,255,0.08)
- Accent: #FF1744
- Success: #00FF41
- Cyan: #00d4ff
