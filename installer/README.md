# HackNow OS — Installer

Calamares installer konfiguratsiyasi. O'zbek tilida lokalizatsiya qilingan.

## Haqida

[Calamares](https://calamares.io) — Linux distributivlari uchun universal installer framework.
HackNow OS uni o'zbek tiliga tarjima qilib, maxsus branding bilan ishlatadi.

## O'rnatish

```bash
sudo apt install calamares calamares-settings-debian
sudo cp -r calamares/* /etc/calamares/
```
